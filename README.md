# Music Player Python

A simple music player built using Python and Pygame.

## Features

- Play MP3 songs
- Pause music
- Resume music
- Stop music
- User-friendly menu

## Technologies Used

- Python 3
- Pygame

## Installation

```bash
pip install pygame
```

## Run

```bash
python music_player.py
```

## Example Menu

```text
1. Play Song
2. Pause Song
3. Resume Song
4. Stop Song
5. Exit
```
